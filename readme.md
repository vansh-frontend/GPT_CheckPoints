## 📖 Overview

RecallGPT is a browser extension built to solve a common problem while using ChatGPT.

When conversations become long during research, learning, coding, or problem-solving, finding previously discussed topics becomes difficult and time-consuming.

RecallGPT allows users to save important prompts as checkpoints and quickly jump back to them without endless scrolling.

---

## ✨ Features

* 📌 Save important prompts as checkpoints
* 🔍 Instantly navigate to saved sections
* 💾 Persistent local storage
* 🗂 Separate checkpoints for each conversation
* 🗑 Delete unwanted checkpoints
* ⚡ Smooth-scroll navigation
* 🎯 Lightweight and easy to use

## 📂 Installation

### Step 1 — Download

Download this repository as ZIP and extract it.

---

### Step 2 — Open Extensions Page

Open Chrome and visit:

```text
chrome://extensions/
```

---

### Step 3 — Enable Developer Mode

Enable the **Developer Mode** toggle from the top-right corner.

---

### Step 4 — Load Extension

Click:

```text
Load unpacked
```

Select the extracted project folder.

---

### Step 5 — Start Using

1. Open ChatGPT
2. Start a conversation
3. Click **Save Checkpoint**
4. Access saved checkpoints from the sidebar
5. Instantly jump back to important prompts

## 🔒 Privacy

RecallGPT does not collect, transmit, or store user data on external servers.

All checkpoint data is stored locally in the browser using the Chrome Storage API.
